# DiscordBot
Simple Discord Bot for My Server

## Requirements

* Node.js 0.10.x or greater

## Token

In the folder of the project create the file _auth.json_.
This file contains the **token** of your bot.

```json
{
    "token": "<your_token>"
}
```

## Install

```
npm install
```

## Run your Bot

```
node bot.js
```
